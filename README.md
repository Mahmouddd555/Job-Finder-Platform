# 💼 Job Search Website

A responsive and interactive **Job Search Web Application** built using **HTML, CSS, and JavaScript**.  
This project simulates a real-world job portal where users can browse jobs, apply, and track applications, while admins can manage job listings.

---

## 🚀 Features

### 👤 User Side
- Browse available job listings
- View detailed job information
- Apply for jobs
- Track applied jobs
- Search functionality for jobs

### 🛠️ Admin Side
- Add new job postings
- Edit existing jobs
- Delete job listings
- View all posted jobs

### 💾 Data Handling
- Uses **Local Storage** for saving:
  - Jobs data
  - User applications
- No backend required

---

## 🧰 Tech Stack

- HTML5
- CSS3 (Responsive Design)
- JavaScript (Vanilla JS)
- Local Storage API
